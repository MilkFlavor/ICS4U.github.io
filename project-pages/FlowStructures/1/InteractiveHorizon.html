🌓 Interactive Horizon

Now I’d like to see an assignment from you that will simulate a landscape with great weather.

In Processing, create a landscape drawing that fulfills the following criteria for a project that may score up to 70%:

  • A canvas of 500 x 500
  • The top half of the canvas should represent the sky
  • The bottom half of the canvas should represent the ground
  • The sketch should start off as a daytime scene
  • If the user hits the “d” key, it should become daytime
  • If the user hits the “n” key, it should become nighttime
  • Use lighter colours for daytime and darker colours for nighttime
  • Each time of day should have at least 3 simple, different objects in the landscape
  • For example, daytime could have a sun, a cloud, and a flower, while nighttime could have a moon, a rock, and a werewolf
  • Don’t overthink the objects; you can be minimal and use just a few shapes to create each object (e.g. a cloud could just be 3 ellipses)

To score up to 100%, additionally include:

  • Additional items which are drawn wherever the mouse is located, when a certain key on the keyboard is pressed (any key of your choice)
  • If the key is pressed while the mouse is in the sky, draw a simple item that would belong in the sky
  • If the key is pressed while the mouse is on the ground, draw a simple item that would belong on the ground
  • The additional items will disappear upon key presses of “d” or “n”
  • The additional items should be different between daytime and nighttime

You variables for values that have significance and are used often (e.g. y-coordinate of the horizon should be a variable).
